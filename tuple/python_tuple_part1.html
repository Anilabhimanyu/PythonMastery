🐍 Python Tuple — Complete Reference

Tuples are immutable sequences used to store ordered collections of items. They’re similar to lists but cannot be changed after creation.

---

📘 Basic Definition

my_tuple = (10, 20, 30)
print(my_tuple)        # (10, 20, 30)
print(type(my_tuple))  # <class 'tuple'>
Note: Single element tuples need a comma — e.g. (10,)
---

💼 Real-world Use Cases

  • Returning multiple values from functions
  • Storing fixed configuration values
  • Dictionary keys for composite identifiers
  • Representing 2D coordinates (x, y)
def minmax(values):
    return (min(values), max(values))

low, high = minmax([10, 20, 30])
print(low, high)   # 10 30
---

🧠 Memory Efficiency

import sys
lst = [1, 2, 3]
tpl = (1, 2, 3)
print(sys.getsizeof(lst), sys.getsizeof(tpl))  # List uses more memory
✅ Tuples are more memory-efficient and faster to iterate than lists.
