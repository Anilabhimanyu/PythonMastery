1 ▸ Set Basics

Definition: An unordered, mutable collection of unique, hashable items.

Set elements (unordered): {10, 'a', 3.14, True} 
# basic set
s = {1,2,3}
print(s)  # {1,2,3} (order may vary)

3 ▸ Accessing Elements

No indexing due to unordered nature. Access via iteration:

s = {10,20,30}
for x in s:
    print(x)

Think of a set as a basket of unique items 🧺. You can add new items, remove items, and check if an item is inside instantly. Duplicates automatically removed.
