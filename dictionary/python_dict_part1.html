1 ‚ñ∏ Dictionary Basics

Definition: A mapping of unique, hashable keys to values. Used for fast lookups (average O(1)).

Keys ‚Üí "name" "age" "city"    Values ‚Üí "Anil" 25 "AP" 
# Basic dict
d = {"name":"Anil", "age":25}
print(d["name"])            # Anil
print(d.get("city","-"))    # - (safe access)

Note: In CPython 3.7+, insertion order is preserved.

9 ‚ñ∏ Dictionary Comprehensions

Powerful and readable way to build dicts.

# basic
squared = {x: x*x for x in range(6)}

# filter
even_sq = {x:x*x for x in range(6) if x % 2 == 0}

# invert mapping (assumes unique values)
inv = {v:k for k,v in d.items()}
10 ‚ñ∏ Advanced Operations & Tools
  • Merging: Python 3.9+: d3 = d1 | d2; older: {**d1, **d2}
  • Unpacking in function calls: fn(**d)
  • collections.defaultdict: auto-init missing keys
  • collections.Counter: frequency counting & helpers
  • Nesting: safely update nested dicts with setdefault() or dict.setdefault(...)
# merge (3.9+)
d3 = d1 | d2

# Counter
from collections import Counter
cnt = Counter(['a','b','a'])  # Counter({'a':2,'b':1})
