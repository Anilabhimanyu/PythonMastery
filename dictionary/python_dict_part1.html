🐍 Python Dictionary — Complete Reference

Dictionaries are mutable, unordered key-value collections in Python. They are widely used for fast lookups, mapping relationships, and structured data storage.

---

📘 Basic Definition

my_dict = {"name": "Anil", "age": 25, "city": "AP"}
print(my_dict)        # {'name': 'Anil', 'age': 25, 'city': 'AP'}
print(type(my_dict))  # <class 'dict'>

💡 Dictionary Comprehension

nums = [1,2,3]
squared = {x: x**2 for x in nums}
print(squared)  # {1:1, 2:4, 3:9}

even = {x: x*2 for x in nums if x%2==0}
print(even)  # {2:4}
---

💼 Real-world Use Cases

  • Counting frequency of elements
  • Storing configuration or settings
  • Fast lookup tables for algorithms
  • Mapping relationships (user → score)
  • Grouping items by property
---

🧠 Memory and Performance

import sys
d = {i:i*2 for i in range(10)}
l = list(range(10))
print(sys.getsizeof(d), sys.getsizeof(l))  # Dict uses more memory, but O(1) lookup
