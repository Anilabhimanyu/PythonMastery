üéØ What is Pandas?

Pandas is a powerful, open-source Python library for data manipulation and analysis. It provides data structures and functions needed to work with structured data seamlessly.

üîë Key Features:

  • DataFrame: 2D labeled data structure (like Excel/SQL table)
  • Series: 1D labeled array (single column)
  • Data Alignment: Automatic and explicit data alignment
  • Missing Data: Integrated handling of missing data
  • Reshaping: Pivot, stack, unstack operations
  • GroupBy: Split-apply-combine operations
  • Merging: Join and merge datasets
  • Time Series: Powerful date/time functionality

üìö Why Use Pandas?

  • Fast and efficient data manipulation
  • Built on NumPy for performance
  • Intuitive syntax similar to SQL/Excel
  • Handles large datasets efficiently
  • Integrates well with other libraries (NumPy, Matplotlib, Scikit-learn)
  • Industry standard for data analysis in Python

üìä Pandas Series

A Series is a one-dimensional labeled array capable of holding any data type.

Creating Series

# From list s = pd.Series([1, 2, 3, 4, 5]) # From list with custom index s = pd.Series([10, 20, 30], index=['a', 'b', 'c']) # From dictionary data = {'a': 10, 'b': 20, 'c': 30} s = pd.Series(data) # From scalar value s = pd.Series(5, index=['a', 'b', 'c']) # From NumPy array s = pd.Series(np.array([1, 2, 3, 4, 5]))

Series Attributes

s = pd.Series([1, 2, 3, 4, 5], index=['a', 'b', 'c', 'd', 'e']) # Basic attributes s.values # Get underlying numpy array s.index # Get index s.dtype # Data type s.shape # Dimensions s.size # Number of elements s.name # Series name s.is_unique # Check if values are unique s.is_monotonic # Check if monotonic

Series Operations

# Accessing elements s['a'] # By label s[0] # By position s[['a', 'c']] # Multiple labels s[0:3] # Slicing # Arithmetic operations (vectorized) s + 10 s * 2 s ** 2 np.sqrt(s) # Boolean operations s > 3 s[s > 3] # Filter # Statistical methods s.sum() s.mean() s.median() s.std() s.var() s.min() s.max() s.quantile(0.75) s.describe() # Summary statistics

Series Methods

# Sorting s.sort_values() # Sort by values s.sort_index() # Sort by index # Unique and value counts s.unique() # Unique values s.nunique() # Count unique values s.value_counts() # Frequency of values # Checking for values s.isin([1, 2, 3]) # Check if values in list s.between(2, 4) # Values between range # Modifying Series s.drop(['a', 'b']) # Drop by label s.drop_duplicates() # Remove duplicates s.replace(1, 100) # Replace values

üìã Pandas DataFrame

A DataFrame is a 2-dimensional labeled data structure with columns of potentially different types.

Creating DataFrames

# From dictionary data = { 'Name': ['Alice', 'Bob', 'Charlie'], 'Age': [25, 30, 35], 'City': ['NYC', 'LA', 'Chicago'] } df = pd.DataFrame(data) # From list of dictionaries data = [ {'Name': 'Alice', 'Age': 25}, {'Name': 'Bob', 'Age': 30} ] df = pd.DataFrame(data) # From 2D NumPy array data = np.array([[1, 2, 3], [4, 5, 6]]) df = pd.DataFrame(data, columns=['A', 'B', 'C']) # From list of lists data = [['Alice', 25], ['Bob', 30]] df = pd.DataFrame(data, columns=['Name', 'Age']) # From Series s1 = pd.Series([1, 2, 3]) s2 = pd.Series([4, 5, 6]) df = pd.DataFrame({'A': s1, 'B': s2}) # Empty DataFrame df = pd.DataFrame() # With specific index df = pd.DataFrame(data, index=['row1', 'row2', 'row3'])

DataFrame Attributes

df = pd.DataFrame({ 'A': [1, 2, 3], 'B': [4, 5, 6], 'C': [7, 8, 9] }) # Basic attributes df.shape # (rows, columns) df.size # Total elements df.ndim # Number of dimensions (2) df.columns # Column labels df.index # Row labels df.dtypes # Data types of columns df.values # Underlying numpy array df.empty # Check if empty # Information methods df.info() # Summary info df.describe() # Statistical summary df.head(n) # First n rows (default 5) df.tail(n) # Last n rows df.sample(n) # Random n rows

Column Operations

# Selecting columns df['A'] # Single column (returns Series) df[['A', 'B']] # Multiple columns (returns DataFrame) # Adding columns df['D'] = [10, 11, 12] # New column df['E'] = df['A'] + df['B'] # Computed column # Renaming columns df.rename(columns={'A': 'Col_A', 'B': 'Col_B'}) df.columns = ['col1', 'col2', 'col3'] # Dropping columns df.drop('D', axis=1) # Drop single column df.drop(['D', 'E'], axis=1) # Drop multiple df.drop(columns=['D', 'E']) # Alternative syntax # Reordering columns df = df[['C', 'A', 'B']]

Row Operations

# Adding rows new_row = pd.DataFrame({'A': [4], 'B': [5], 'C': [6]}) df = pd.concat([df, new_row], ignore_index=True) # Dropping rows df.drop(0) # Drop by index df.drop([0, 1]) # Drop multiple df.drop(index=[0, 1]) # Alternative # Resetting index df.reset_index(drop=True) # Reset to default df.reset_index() # Keep old index as column # Setting index df.set_index('A') # Set column as index
